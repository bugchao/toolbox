# Timezone Converter

A React component for comparing multiple timezones and converting meeting times.

## Features
- Real-time timezone comparison
- Meeting time conversion across zones
- Responsive design
- Lucide React icons

## Usage
```jsx
import TimezoneConverter from './TimezoneConverter'

function App() {
  return (
    <TimezoneConverter />
  )
}
```

## Dependencies
- React
- Lucide React
- Moment.js

## Installation
```bash
npm install
```

## Development
```bash
npm start
```
