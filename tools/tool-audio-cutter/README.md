# 音频剪辑工具 (Audio Cutter)

在线音频裁剪工具，支持多种音频格式，可视化波形编辑。

## 功能特点

- ✅ 支持多种音频格式（MP3/WAV/AAC/OGG）
- ✅ 支持拖拽上传
- ✅ 音频波形可视化
- ✅ 拖拽选择裁剪区域
- ✅ 实时预览裁剪部分
- ✅ 显示时间轴和时长
- ✅ 支持精确时间输入
- ✅ 支持下载裁剪后的音频
- ✅ 纯前端实现，数据不上传

## 技术实现

- WaveSurfer.js - 波形可视化
- Web Audio API - 音频处理
- React + TypeScript
- Tailwind CSS

## 使用方法

1. 上传或拖拽音频文件
2. 在波形图上拖拽选择裁剪区域
3. 或使用精确时间输入框设置开始/结束时间
4. 点击"剪辑并下载"按钮
5. 下载裁剪后的音频文件（WAV格式）

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build
```

## 注意事项

- 输出格式为 WAV（未压缩，音质最佳）
- 所有处理在浏览器本地完成，不上传服务器
- 大文件可能需要较长处理时间
