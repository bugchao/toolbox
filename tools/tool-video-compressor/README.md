# 视频压缩工具 (Video Compressor)

在线视频压缩工具，支持多种视频格式，可自定义压缩质量和目标大小。

## 功能特性

- ✅ 支持多种视频格式（MP4, AVI, MOV, MKV, WebM）
- ✅ 拖拽上传视频文件
- ✅ 三种压缩质量选项（高/中/低）
- ✅ 可选目标文件大小
- ✅ 实时显示压缩进度和预估时间
- ✅ 原视频和压缩后视频对比预览
- ✅ 显示文件大小和压缩率
- ✅ 支持下载压缩后的视频
- ✅ 纯前端实现（使用 FFmpeg WebAssembly）

## 技术实现

- **FFmpeg.wasm**: 纯前端视频压缩，无需后端服务器
- **React**: 组件化开发
- **TypeScript**: 类型安全
- **Tailwind CSS**: 响应式设计
- **i18n**: 中英文双语支持

## 使用方法

1. 点击或拖拽上传视频文件（最大 500MB）
2. 选择压缩质量（高/中/低）
3. 可选：设置目标文件大小
4. 点击"开始压缩"
5. 等待压缩完成
6. 预览对比原视频和压缩后视频
7. 下载压缩后的视频

## 压缩质量说明

- **高质量**: CRF 23, 文件较大，画质最好
- **中等质量**: CRF 28, 推荐选项，平衡画质和文件大小
- **低质量**: CRF 35, 文件最小，画质较低

## 注意事项

- 视频压缩在浏览器中进行，不会上传到服务器
- 首次使用需要加载 FFmpeg（约 30MB）
- 压缩时间取决于视频大小和设备性能
- 建议使用现代浏览器（Chrome, Firefox, Edge）
