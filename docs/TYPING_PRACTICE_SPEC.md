## 二、核心功能

### 1. 多种练习模式

#### 1.1 速度测试模式
- **功能**：测试打字速度（WPM - Words Per Minute）
- **时长选项**：30秒、60秒、120秒、自定义
- **文本来源**：
  - 随机英文文章
  - 随机中文文章
  - 经典名言
  - 编程代码片段
- **实时反馈**：
  - 当前速度（WPM）
  - 准确率（%）
  - 已用时间
  - 进度条

#### 1.2 指法练习模式
- **基础指法**：
  - 主键位练习（ASDF JKL;）
  - 上排键位（QWER UIOP）
  - 下排键位（ZXCV NM,.)
  - 数字键位（1234567890）
- **进阶练习**：
  - 符号键位（!@#$%^&*()）
  - 组合键练习（Shift + 字母）
  - 常用单词练习
- **手指提示**：显示应该用哪个手指按键

#### 1.3 文章练习模式
- **文章分类**：
  - 短文（100-200字）
  - 中文（300-500字）
  - 长文（500-1000字）
- **难度等级**：
  - 简单：常用字词
  - 中等：混合词汇
  - 困难：专业术语
- **自定义文本**：用户可以粘贴自己的文本练习

#### 1.4 单词练习模式
- **英文单词**：
  - 常用1000词
  - 四六级词汇
  - 托福/雅思词汇
  - 编程关键词
- **中文词组**：
  - 常用成语
  - 诗词名句
  - 专业术语
- **随机生成**：每次练习随机抽取单词

#### 1.5 代码练习模式
- **编程语言**：
  - JavaScript
  - Python
  - Java
  - C/C++
  - HTML/CSS
- **代码类型**：
  - 函数定义
  - 循环语句
  - 条件判断
  - 常用算法
- **语法高亮**：代码显示带语法高亮

#### 1.6 游戏模式
- **打字赛车**：速度越快，赛车跑得越快
- **打字射击**：正确打字消灭目标
- **打字跑酷**：打字控制角色跳跃
- **排行榜**：本地记录最高分

### 5.2 核心算法

#### WPM 计算
```typescript
// WPM = (正确字符数 / 5) / (时间(秒) / 60)
const calculateWPM = (correctChars: number, timeInSeconds: number): number => {
  if (timeInSeconds === 0) return 0;
  return Math.round((correctChars / 5) / (timeInSeconds / 60));
};
```

#### 准确率计算
```typescript
// 准确率 = (正确字符数 / 总输入字符数) * 100
const calculateAccuracy = (correctChars: number, totalChars: number): number => {
  if (totalChars === 0) return 100;
  return Math.round((correctChars / totalChars) * 100);
};
```

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
