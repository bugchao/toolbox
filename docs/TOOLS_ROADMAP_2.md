## 一、分类目录（固定，后续规划按此填写）

| 分类 ID | 分类名称     | 说明 |
|--------|--------------|------|
| `utility` | 实用工具     | 日常效率：图片/文档/表格/格式转换、生成器等 |
| `dev`   | 研发工具     | 开发调试：编解码、正则、时间戳、哈希、代码格式化等 |
| `design` | 设计工具     | UI/UX、颜色、字体、图标等设计辅助工具 |
| `marketing` | 营销工具   | 内容创作、SEO、文案生成等 |
| `entertainment` | 娱乐工具 | 游戏、趣味生成器等 |
| `education` | 教育工具   | 语言学习、数学工具等 |
| `business` | 商务工具    | 发票、合同、报价等 |
| `life`    | 生活工具       | 效率习惯、财务消费、健康生活、日常工具等 |

新增规划时，**必须**指定上述其一作为「分类」。
